Simplify

A mathematical tool which solves and simplifies propositional logic expressions

Python
Simplify
I programmed majority of the backend for this tool.

Firstly, I focused on transforming propositional logic statements into truth tables for the user to optionally view.
Secondly, fully simplifying expressions into conjunctive and disjunctive form. This would then be the answer to the question box which the user has to fill in.

The tool has two types of questions. Either, you could make a txt file with the questions and the tool would read from the file and display those questions to the user. Or the user could enter their own question and attempt it themselves.

Tetris AI

A program that calculates the best possible move to play in a game of Tetris

Python
Tetris AI
When calculating the best possible move, there are 40 possibilities. The programme has internal factors which weight the strength of each move. For example, how many holes would a move create? Or the aggregate height that adding a block would have? The programme would internally clone the game board, then try out all 40 possible move combinations. It will assess the strength of each move according to the internal weighting factors. I was manually tuning the importance of these factors, so some factors would be given priority over others.

Note App

A note taking web application to store notes


Java
Note App
This is an extensive web app which gives the user options to make their own notes, save notes, edit notes and rename notes. All notes consist of a title and a body and can contain all characters. I store every note in a text file where each note is separated by special characters. These special characters allow the program to detect when a note finishes and when the next note starts!
